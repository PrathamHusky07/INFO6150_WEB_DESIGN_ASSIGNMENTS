# INFO_6105_WEB_DESIGN_ASSIGNMENTS

Part A
Favicon and CSS files are linked using the link tag.The navbar is oriented using flexbox and is covered by the nav tag.The navbar has a sign-up button that, when clicked, goes to a sign-up form and a logo that, when clicked, takes users to the home page.The form tag is used to create the sign-up form. There are two input fields: one for passwords and the other for emails. When the form's submit button is clicked, the form is submitted and the user input is shown on the url.Table tag is used to create the menu on the main page. The price of each type of coffee available on the Tatte's website is listed on the table.
